<h1 align="center">📊 Customer Churn Analysis</h1>

<p align="center">
Projeto de análise de evasão de clientes (Churn) utilizando Python e Pandas.
</p>

<hr>

<h2>🎯 Objetivo do Projeto</h2>
<p>
Realizar a extração, tratamento e análise exploratória de dados de clientes para identificar padrões relacionados à evasão (Churn), gerando insights estratégicos para retenção.
</p>

<hr>

<h2>🚀 Etapas do Projeto</h2>

<h3>1️⃣ Extração de Dados</h3>
<ul>
  <li>Carregamento dos dados diretamente via API utilizando Python</li>
  <li>Conversão para DataFrame com Pandas para facilitar a manipulação</li>
</ul>

<h3>2️⃣ Conhecendo o Dataset</h3>
<ul>
  <li>Exploração das colunas e tipos de dados</li>
  <li>Análise do dicionário de dados</li>
  <li>Identificação das variáveis mais relevantes para churn</li>
</ul>

<h3>3️⃣ Verificação e Tratamento de Inconsistências</h3>
<ul>
  <li>Identificação de valores ausentes, duplicados e erros de formatação</li>
  <li>Correção e padronização dos dados</li>
</ul>

<h3>4️⃣ Engenharia de Variáveis</h3>
<ul>
  <li>Criação da coluna <strong>Contas_Diarias</strong> com base no faturamento mensal</li>
  <li>Transformação de variáveis categóricas (ex: Sim/Não → 1/0)</li>
  <li>Renomeação e padronização de colunas</li>
</ul>

<hr>

<h2>📈 Análise Exploratória de Dados</h2>

<ul>
  <li>Análise descritiva (média, mediana, desvio padrão)</li>
  <li>Distribuição da variável Churn</li>
  <li>Churn por variáveis categóricas (gênero, contrato, pagamento, etc.)</li>
  <li>Análise de correlação entre variáveis</li>
</ul>

<p>
Foram utilizados gráficos para apoiar a interpretação dos dados e identificar padrões relevantes.
</p>

<hr>

<h2>📊 Principais Análises Realizadas</h2>
<ul>
  <li>Relação entre valor da conta diária e evasão</li>
  <li>Impacto do tipo de contrato no churn</li>
  <li>Influência da quantidade de serviços contratados</li>
  <li>Correlação entre variáveis numéricas</li>
</ul>

<hr>

<h2>📝 Relatório Final</h2>
<p>
O notebook inclui um relatório estruturado contendo:
</p>

<ul>

  <li><strong>Limpeza e Tratamento:</strong> Ajustes realizados nos dados</li>
